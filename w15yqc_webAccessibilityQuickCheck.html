A Web Accessibility Quick Check Protocol

The following protocol (or procedure) uses the W15yQC extension to check a web page for obvious accessibility issues.

Appropriate alternative text for images.

Having the appropriate replacement text for images is very important for non-visual users, and other users of screen-reading technologies. There are two basic tests to make for each image:

  • If the image serves a meaningful (non-redundant) purpose, it should have appropriate and concise alternative text supplied for it.
  • If the image is strictly decorative (or redundant), then it should be indicated as such so it won't be announced by assistive technologies.

How to check for appropriate alternative text using W15yQC:

  • On the W15yQC Toolbar, choose "Inspectors" and then "Images"
  • Scroll through each image and verify the appropriate alternative text indication was supplied:
    • By default, W15yQC will attempt to scroll to and highlight the selected image.
    • Determine if the image is meaningful and non-redundant.
      • If yes (image is meaningful):
        • Does the text listed in the effective label column accurately and concisely describe the image?
      • If no (image is decorative or redundant):
        • Has the image been marked so it will not be announced? This can be done by:
          • Supplying an empty alternative text (alt="" or aria-label="").
          • Marking the image with a role of 'presentational'.
