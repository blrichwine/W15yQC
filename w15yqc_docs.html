W15yQC Features

  • Generate report with list of potential common Web accessibility issues for dynamically generated Web pages.
  • Web site scanner to help find pages to check for accessibility issues.
  • A Semantic View / intellegent Remove Styles View tool to see what Web pages look like to non-visual users.
  • Inspector Dialogs to check page features for accessibility issues.
  • Invalid ID Checker to inspect Web pages for non-unique and invalid IDs.
  • Color Contrast (Luminosity) Checker to inspect Web pages for potential color contrast issues.
  • Color Contrast (Luminosity) Tool to pick up to 3 simultaneous colors that meet contrast requirements with each other.
  • Focus Tracker Tool that attempts to keep keyboard focus visible so pages without visible focus indication can be tested for other keyboard accessibility issues.
